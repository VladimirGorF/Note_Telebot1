Это учебный проект: 
Мини-записная книжка на базе SQL Lite c использованием функций телеграмм-бота для CRUD операций.
Для запуска сервера используется файл main.py, в котором необходимо в поле API_TOKEN = 'XXXXXXXXXXXXXXXXXXXXXXXX' вести свой токен, который выдает BOT-FATHER в Телеграмм. 

Бот предлагает ограниченный функционал в виде команд:
showAll,
search,
delete,
change,
add.

Все команды из телеграмм-бота отрабатываются языком пайтон и с помощью команд SQL реализовываются в базе данных, которая находится здесь же в файле data_base.py.


Также перед запуском программы необходимо использовать Necesarry Libs.
Use Necesarry Libs to install TelegramBotAPI
